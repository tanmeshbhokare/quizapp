# quizapp
End to End CICD pipeline for simple WebApp

![image](https://user-images.githubusercontent.com/42235943/180472508-fae889f1-cf66-48bc-9cf9-9f6be5e5e523.png)

Pre-Requisite
On Ubuntu/Any Linux
- Install Git
- Install Java 17
- Install Jenkins
- Install Gitea
- Install SonarQube
- Install Artifactory
- Install Ansible
- Install Docker and Docker compose
- Install Prometheus
